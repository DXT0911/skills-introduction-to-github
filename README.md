# Python 从入门到精通教程

_一个完整的 Python 编程教学代码仓库，帮助初学者系统学习 Python。_

## 欢迎 👋

本仓库是一个面向 Python 初学者的完整教学资源，内容由浅入深，涵盖从基础语法到高级特性的方方面面。无论你是编程零基础，还是有一定经验想要系统学习 Python，这里都有适合你的内容。

- **适合人群**：编程初学者、想学 Python 的开发者、在校学生
- **学习内容**：Python 基础语法、数据结构、面向对象编程、高级特性、实战项目
- **前置要求**：无，只需要安装 Python 3.x 即可
- **学习时长**：可按章节自主安排，建议每天 1-2 小时，约 4-6 周完成

## 如何使用本仓库

1. **安装 Python**：前往 [python.org](https://www.python.org/downloads/) 下载安装 Python 3.x
2. **克隆仓库**：
   ```bash
   git clone https://github.com/DXT0911/skills-introduction-to-github.git
   cd skills-introduction-to-github
   ```
3. **运行示例**：
   ```bash
   python 01-basics/01-hello-world.py
   ```
4. **按顺序学习**：建议从第一章开始，按文件编号顺序学习

## 学习建议

- **阅读代码**：每个文件都有详细的注释，先阅读理解
- **动手实践**：修改示例代码，尝试不同的输入和场景
- **思考练习**：每节课后有练习题，尝试自己解决
- **反复回顾**：遇到不懂的概念，多看几遍，结合实例理解

## 环境要求

- Python 3.8 或更高版本
- 推荐使用 VS Code 或 PyCharm 作为编辑器
